t # AgentSpace

AgentSpace is a modular, scalable AI agent marketplace and execution platform built with Next.js, ShadCN, and Genkit.

## Features

- **Explore Marketplace**: Discover high-quality AI agents across various categories.
- **Agent Execution**: Run agents directly with real inputs and structured outputs.
- **Battle Mode**: Compare two agents side-by-side to evaluate performance.
- **Auto Repo Builder**: Generate agent configurations from natural language descriptions.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **UI**: React, ShadCN UI, Tailwind CSS
- **AI**: Genkit (Google Gemini)
- **Backend**: Firebase Firestore & Auth

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/goblinasaddy/AgentSpace.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env` file and add your `GEMINI_API_KEY`.

4. **Run the development server**:
   ```bash
   npm run dev
   ```
